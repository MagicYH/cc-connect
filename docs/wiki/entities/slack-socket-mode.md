# Slack Socket Mode

Slack's Socket Mode WebSocket connection plus REST API at `slack.com`.

## Project Relation

Platform adapter in `platform/slack/` using `slack-go/slack` (v0.16.0). Supports message threads for session tracking.

Config: `type="slack"`.
