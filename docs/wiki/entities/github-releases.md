# GitHub Releases API

GitHub Releases API at `api.github.com/repos/chenhg5/cc-connect/releases`.

## Project Relation

Used for auto-update checks and version info in `core/updater.go` and `cmd/cc-connect/update.go`. Also serves remote provider presets and skill presets JSON from `raw.githubusercontent.com`.
