# DingTalk Stream

DingTalk's Stream mode SDK and REST API at `api.dingtalk.com`.

## Project Relation

Platform adapter in `platform/dingtalk/` using `open-dingtalk/dingtalk-stream-sdk-go` (v0.9.1). Supports AI cards, file download/upload, and reactions.

Config: `type="dingtalk"`.
