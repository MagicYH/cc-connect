# Tailwind CSS

Tailwind CSS (v3.4.17) is the utility-first CSS framework used in cc-connect's web dashboard frontend.

It is paired with `@tailwindcss/typography` (v0.5.19) for prose styling of markdown-rendered content. PostCSS and autoprefixer handle CSS processing.
