# React

React (v19.1.0) is the UI framework for cc-connect's web dashboard frontend, located in `web/`.

The dashboard provides a browser-based interface for monitoring and managing cc-connect sessions. It uses React Router for navigation, Zustand for state management, and Tailwind CSS for styling.
