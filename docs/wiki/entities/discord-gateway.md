# Discord Gateway

Discord's WebSocket Gateway and REST API at `discord.com`.

## Project Relation

Platform adapter in `platform/discord/` using `bwmarrin/discordgo` (v0.29.0). Supports slash commands, threads, and reactions.

Config: `type="discord"`.
