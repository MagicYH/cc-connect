# ffmpeg

Audio/video processing CLI tool, auto-detected on PATH.

## Project Relation

Used for audio format conversion in STT (AMR/OGG to MP3) and TTS (WAV to Opus) pipelines in `core/speech.go` and `core/tts.go`.
