# LINE

LINE is a messaging platform popular in Japan, Taiwan, and Southeast Asia. In cc-connect, it is one of the supported messaging platforms (`platform/line/`).

The LINE adapter uses the `line/line-bot-sdk-go/v8` (v8.19.0) library for the LINE Messaging API.
