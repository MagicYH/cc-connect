# Claude Code

Anthropic's CLI coding agent. One of 14 supported AI agent adapters in CC-Connect.

**In this project**: The `agent/claudecode/` package (3,631 lines) implements the adapter. Key files: `claudecode.go` (1,464 lines), `session.go` (1,024 lines), `cc_hooks.go` (347 lines), `claude_usage.go` (663 lines). Implements `AgentDoctorInfo` (CLI binary: `claude`, display name: `Claude`).
