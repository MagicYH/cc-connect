# DashScope / Qwen

Alibaba Cloud's DashScope API at `dashscope.aliyuncs.com` for Qwen models.

## Project Relation

Provides STT (Qwen ASR) via `core/speech.go` and TTS (Qwen TTS) via `core/tts.go`.

Config: `[speech] provider="qwen"`, `[tts] provider="qwen"`.
