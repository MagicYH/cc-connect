# Charmbracelet

Charmbracelet is the company/project behind the terminal UI toolchain used in cc-connect's CLI. Three of their libraries are used:

- **bubbletea** (v1.3.10) — TUI framework for the sessions overview screen (`cmd/cc-connect/sessions_tui.go`)
- **bubbles** (v1.0.0) — TUI component library (companion to bubbletea)
- **lipgloss** (v1.1.0) — Terminal styling and layout
