# DingTalk

DingTalk is a business communication and collaboration platform by Alibaba. In cc-connect, it is one of the supported messaging platforms (`platform/dingtalk/`).

The DingTalk adapter uses the `open-dingtalk/dingtalk-stream-sdk-go` (v0.9.1) library for the DingTalk Stream API.
