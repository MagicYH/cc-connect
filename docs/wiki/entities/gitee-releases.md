# Gitee Releases API

Gitee Releases API at `gitee.com/api/v5/repos/cg33/cc-connect/releases`.

## Project Relation

Automatic fallback for GitHub when unreachable (China users). Used for auto-update checks and version info. Also serves provider presets and skill presets as fallback.
