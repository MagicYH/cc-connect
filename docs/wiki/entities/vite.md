# Vite

Vite (v6.3.2) is the frontend build tool and dev server for cc-connect's web dashboard, configured in `web/`.

It provides fast hot-module replacement during development and optimized production builds. The `@vitejs/plugin-react` (v4.4.1) plugin enables React support.
