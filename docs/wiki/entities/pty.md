# pty (creack/pty)

Unix pseudo-terminal Go library (`github.com/creack/pty` v1.1.24).

## Project Relation

Used across agent packages (`agent/claudecode/`, `agent/codex/`, etc.) to spawn agent CLI subprocesses with proper terminal I/O. Enables interactive session management with full PTY support.
