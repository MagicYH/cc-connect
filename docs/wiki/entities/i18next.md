# i18next

i18next (v25.1.2) is an internationalization framework used in cc-connect's web dashboard, paired with `react-i18next` (v15.5.1) for React integration.

This mirrors the backend's i18n system (`core/i18n.go`), ensuring consistent multi-language support across both the API and the web UI.
