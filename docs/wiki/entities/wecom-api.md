# WeChat Work / WeCom API

WeChat Work (WeCom) enterprise messaging API at `qyapi.weixin.qq.com`.

## Project Relation

Platform adapter in `platform/wecom/` with custom HTTPS callback and WebSocket implementation. Supports file/image download and Markdown messages. API base URL is configurable via `api_base_url`.

Config: `type="wecom"`.
