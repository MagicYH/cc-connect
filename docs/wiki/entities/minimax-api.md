# MiniMax API

MiniMax AI API at `api.minimaxi.com` / `api.minimax.io`.

## Project Relation

Provides TTS (`core/tts.go`) and serves as an OpenAI-compatible agent provider for third-party model routing.

Config: `[tts] provider="minimax"`, `[projects.agent.providers] name="minimax"`.
