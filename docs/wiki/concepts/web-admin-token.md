# Web Admin Token

Authentication token for the web management API. Generated by `/web setup` command and sent to the user via the IM platform reply in plaintext.

Risk: anyone in the chat channel can see the token and gain full admin access (project settings, disabled commands, mode, agent type). No per-user authorization — all bridge users share a "web-admin" identity.

Project relation: `core/engine.go`, `core/i18n.go`, `core/bridge.go`.
