# Provider Preset

Remote JSON configuration that defines available AI model providers and their connection settings.

## Project Relation

Fetched from GitHub Raw (`provider-presets.json`) with Gitee as fallback. Managed in `core/provider_presets.go`. Allows adding new provider configurations without code changes. Configurable via `provider_presets_url`.
