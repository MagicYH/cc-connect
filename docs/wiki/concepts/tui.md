# TUI (Terminal User Interface)

The Terminal User Interface is cc-connect's in-terminal session management screen, implemented using the Charmbracelet toolchain (bubbletea, bubbles, lipgloss).

Located in `cmd/cc-connect/sessions_tui.go`, it provides an interactive overview of active sessions directly in the terminal without requiring the web dashboard.
