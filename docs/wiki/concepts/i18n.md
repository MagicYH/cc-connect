# i18n (Internationalization)

cc-connect implements internationalization at two levels:

- **Backend** (`core/i18n.go`): All user-facing strings use `MsgKey` constants with translations for EN, ZH, ZH-TW, JA, ES. Access via `e.i18n.T(MsgKey)` or `e.i18n.Tf(MsgKey, args...)`.
- **Frontend** (`web/`): Uses `i18next` (v25.1.2) with `react-i18next` (v15.5.1) for React integration.

Both systems must be updated when adding user-facing strings.

Cross-references: [tech-stack](../sources/tech-stack.md), [i18next](../entities/i18next.md), [project-structure](../sources/project-structure.md)
